# cellulose

`cellulose` is the planned YggdrasilHQ spreadsheet and analytical workspace.

The product direction is intentionally ambitious: not a toy grid, but an AI-first environment for serious spreadsheet-style work with strong automation, observability, and demoable release quality from the start.

## Positioning

The current working mental model is:

- spreadsheet-native interaction
- serious modeling workflows
- AI-assisted analysis and automation
- reusable `yggui` app-control, observability, and changelog-demo infrastructure

## Repository Scope

This repository starts as product and architecture scaffolding rather than an implementation-heavy codebase.

The first durable work here should be:

- product definition
- architecture choices
- shared `yggui` platform assumptions
- proof-bundle and release-storytelling expectations
